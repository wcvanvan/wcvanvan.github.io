**wcRay** is a software path tracing program, implemented in modern C++. 

It depends on no other libraries except stb_image to read in images.

## List of Features

- **Importance Sampling**
- **Materials**
    - Lambertian
    - Metal
    - Dielectric
    - Participating Media
- **Camera Functionalities**
    - Defocus Blur
    - Motion Blur
- **Acceleration Structure**
    - BVH
- **Texture**
    - Perlin Noise
    - Checkerboard
    - Texture Mapping
